Open Paws: From Software Engineer to Local Director of Open Paws India

How consistent shipping — RAG systems, data pipelines, hackathon wins — led to co-owning the Indian entity of a global open-source animal advocacy org.

Open PawsLeadershipOpen SourceIndia

What Open Paws Does

Open Paws builds AI tools for animal advocacy — prediction models, knowledge graphs, vector databases, speciesist framing metrics, structured coding instruction sets for 12+ AI tools (137 files), and a CryptPad fork for strategic project management. Fully open source, global contributors, serious production tooling.

My Technical Contributions

AfA Resource Chatbot (RAG)

Built a Retrieval-Augmented Generation system using OpenAI embeddings, dual vector storage (Pinecone + Supabase pgvector), and conversational retrieval. Makes the entire Open Paws knowledge base instantly searchable.

Project Slingshot UK

Automated MEL dashboard tracking UK factory farming discourse. 12+ API data sources (media, parliamentary, social). Gemini 2.0 Flash for analysis. Supabase data layer. Full stakeholder docs.

Code4Compassion Hackathons

Mumbai (Winner): Built AFOG → Open Permit. Berlin: Built LEWS. Both under the Open Paws hackathon series.

Infrastructure & Tooling

Structured-coding-with-ai repo contributions. Supabase alternatives evaluation. Platform infrastructure across multiple repos.

Open Paws India Pvt Ltd

Currently being registered. Open Paws founder as primary shareholder/director. Me as second shareholder and Local Director. The Indian entity handles all India-based operations: payments (Wise integration), hiring (20 Adventurers Guild interns), vendor relationships, GST/TDS compliance, and partnerships.

Why India needs a local entity: Payment processing (Indian banking regs), hiring compliance, tax compliance, and partnership legitimacy with Indian NGOs and government bodies.

Adventurers Guild Integration

The Guild operates under Open Paws, creating a symbiotic loop. Open Paws provides brand, payment infrastructure, legal framework, client relationships. The Guild provides developer talent pipeline, bootcamp students, real-world project capacity.

Two-track system: 20 quality interns handle D+ rank work (backlog, features, maintenance). Bootcamp students handle F/E rank tasks (learning, guided projects). QA team: Open Paws founder + me + a mentor being hired + 1 senior dev. All under Open Paws India once registration completes.

How Claude Code Supports This

Legal research for MCA filings, DIN registration, and shareholding agreements. Financial operations — Wise payment flows, Indian banking integration, tax planning, compensation structures for Indian market rates. Every operational document — intern onboarding, IP/legal frameworks, Adventurers Guild integration plans — developed through Claude Code sessions.

Key Lesson

Ownership is earned through output, not titles. The RAG chatbot, data pipelines, hackathon wins, infrastructure work — each contribution built trust. In open source orgs, the path from "contributor" to "co-owner of the entity" is paved by consistently shipping real value.
