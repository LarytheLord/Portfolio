Philosophy

OSS contributions are the highest-signal portfolio item a developer can have. Anyone can build a todo app. Not everyone can get a PR merged into a repo with 3.6k stars. Maintainers don't care about your resume — they care about whether your code solves a real problem and passes their standards.

I target repos with 1k+ stars, active maintenance, real production users, and meaningful issues. Focus areas: sustainability tech, scientific computing, developer infrastructure — domains where the impact compounds.

How Claude Code Accelerates OSS Work

The bottleneck in OSS contribution isn't coding — it's understanding unfamiliar codebases quickly enough to contribute meaningfully. Claude Code sessions handle:

Codebase Understanding

Large repos (nextflow, scanpy) have complex architectures. Claude Code helps quickly understand module relationships, build systems, and contribution guidelines.

PR Preparation

Drafting clear PR descriptions, commit messages, and change justifications that align with each repo's conventions.

Review Iteration

When maintainers request changes, Claude Code helps iterate quickly — understanding their feedback and adapting the code accordingly.
