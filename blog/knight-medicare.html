Knight Medicare: AI-Powered CBT Therapy Platform

Building accessible mental health infrastructure with a Harvard PhD psychologist. 27 disorders, 10 therapy modalities, Next.js 15, React 19, Supabase. Serving as CTO.

HealthTech AI/ML Startup

The Healthcare Gap

We are facing a mental health accessibility crisis. Traditional therapy is expensive, time-intensive, and has limited reach. Our vision was to create an AI-assisted therapy platform to bridge this gap.

Building with Experts

We collaborated with a Harvard PhD psychologist to ensure clinical rigor. As CTO, my role was translating this clinical expertise into a technical product, balancing the need for strict adherence to therapy protocols with a seamless user experience.

Technical Foundation

Architecture

Next.js 15 + React 19 with TypeScript. Server components for fast loads, streaming for therapeutic conversation interfaces. Supabase for auth, real-time, and PostgreSQL.

Coverage

27 psychiatric disorders — from depression and anxiety to PTSD, OCD, eating disorders, and substance use. 10 therapy modalities including CBT core, behavioral activation, exposure therapy, and mindfulness.

Safety

Crisis detection and escalation pathways. Session logging for clinical review. Explicit disclaimers and therapist referral triggers. Abstraction layers built after the Supabase India block.

How Claude Code Runs the Technical Side

The full Next.js 15 / React 19 codebase was built through Claude Code sessions. The therapeutic conversation engine, Supabase schema design, auth flows, and abstraction layers — all developed collaboratively. When the Supabase India block hit, Claude Code designed abstraction layers so swapping providers requires changing adapter code, not business logic. The 27×10 disorder-modality data model, therapy session state machines, progress tracking structures — all designed through Claude Code.

The Product Vision

The platform features therapist-client matching algorithms, automated administrative tasks (scheduling, notes, insurance), and AI-powered therapy insights like sentiment analysis and progress tracking.

Lessons in Health Tech

  • Navigating regulatory complexity.
  • Building trust in sensitive domains.
  • Balancing innovation with clinical validation.
  • The critical importance of domain expert partnerships.
